## How to Fetch

```bash
curl -s https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/<brand>/DESIGN.md
```

## How to Use

1. Fetch the DESIGN.md for the target brand
2. Extract tokens: colors, typography, spacing, components
3. Apply to the project's CSS (see `references/apple-design-system.md` for a worked example)
4. Create a project-specific DESIGN.md adapted from the reference
