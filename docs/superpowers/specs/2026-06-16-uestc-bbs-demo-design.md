# 成电Wiki论坛演示站 设计文档

> 基于 `wiki.bbs.uestc.net_品牌设计小组汇报方案` 制作的纯静态演示 demo。
> 日期：2026-06-16

## 1. 目标与定位

- **定位**：混合 —— 顶部品牌介绍，中部模拟 BBS 首页（板块 + 热门帖），底部品牌故事/愿景。一页滚动覆盖「首页 + 介绍」。
- **用途**：演示 demo，不做真实登录/发帖/后端。
- **技术**：纯静态 `index.html`，内联 CSS + 原生 JS，零依赖、零构建、零 CDN。

## 2. 配色与视觉

严格采用文档定义的成电蓝白主色：

| 角色 | 色值 | 用途 |
|---|---|---|
| 成电蓝（主色）| `#003366` | 导航栏、Hero 背景、标题、主按钮、强调 |
| 深蓝（渐变端）| `#001f3f` | Hero 渐变深处，营造纵深 |
| 亮蓝（点缀）| `#1e6bb8` | 悬停态、链接、icon 高亮 |
| 白 | `#ffffff` | 卡片、Hero 文字 |
| 浅灰 | `#f5f5f5` | 分区背景 |
| 科技银 | `#e0e0e0` | 边框、分隔线、卡片描边 |

装饰符号：代码线条 `</>`、对话气泡、电路纹理 —— 用 SVG/CSS 绘制进 Hero 背景，不引外部图片。

字体：系统无衬线栈（PingFang SC / Microsoft YaHei / -apple-system / Segoe UI / sans-serif）。

## 3. 页面结构（单页滚动）

```
① 顶部导航（fixed，毛玻璃，滚动变实色）
   Logo「成电Wiki论坛」+ 锚点（首页/板块/故事/展望）+ 域名

② Hero 英雄区
   - 成电蓝渐变背景 + 浮动代码符号/气泡粒子（CSS keyframes）
   - 大标题「成电Wiki论坛」+ 副标题「成电学子专属成长互助开源社区」
   - 域名 wiki.bbs.uestc.net
   - 按钮：进入论坛（锚点→板块区）/ 了解我们（锚点→故事区）
   - 打字机轮播广告语

③ 数据条（数字滚动动画，进入视口触发）
   80% 在校生覆盖 · 5000+ 校友 · 5 大板块 · 7 位创立者

④ BBS 首页模拟区（核心演示）
   - 5 个板块卡片：升学 / 竞赛 / 职业 / 生活 / 校友专区（悬停上浮+发光）
   - 「热门讨论」帖子列表（模拟数据：板块标签、标题、作者、回复数、热度）
   - 右侧「社区动态」侧栏（新帖/活跃用户，纯展示）

⑤ 品牌介绍区（4 个价值观卡片）
   资源互助 / 平等开放 / 真诚分享 / 温暖陪伴，滚动渐入

⑥ 品牌故事区
   竖向时间轴：7 名学子创立 → 打破壁垒 → 轻实名 → 开源共建

⑦ IP 形象 + 未来展望
   「电小Wiki」IP 卡片（CSS/SVG 简约头像）+ 三阶段目标（短期/中期/长期）

⑧ 页脚
   广告语、域名、价值观口号、版权占位
```

## 4. 动效清单（纯 CSS/JS，无库）

- **Hero 粒子背景**：绝对定位元素 + keyframes 缓慢浮动飘移
- **打字机广告语**：JS 逐字输出/删除，循环 3 条广告语
- **滚动渐入**：IntersectionObserver，元素进视口 `opacity 0→1` + `translateY(30px→0)`
- **数字滚动**：数据条进入视口时从 0 滚到目标值（requestAnimationFrame）
- **卡片悬停**：`translateY(-8px)` + 阴影加深 + 蓝色边框发光
- **导航毛玻璃**：`backdrop-filter: blur()`，滚动加 `.scrolled` 类变实色
- **板块卡片渐显**：依次 `transition-delay`
- **时间轴**：竖线 + 节点圆点，滚动时从上往下「画」出来（scaleY 0→1）

## 5. 响应式断点

- 桌面（≥1024px）：板块 5 列、帖子+侧栏双栏
- 平板（768–1023px）：板块 2-3 列、侧栏收起到帖子下方
- 手机（<768px）：全部单列，导航折叠成汉堡菜单
